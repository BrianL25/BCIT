HTML < blockquote > Tag

Definition, Usage, and Semantic Value(s)

The < blockquote > tag specifies a section that is quoted from another source. Browsers usually indent < blockquote > elements.

< blockquote > has a "cite" attribute that specifies the source of the quotation.

Example:

A quote from the official webpage of the International Monetary Fund:

The IMF, also known as the Fund, was conceived at a UN conference in Bretton Woods, New Hampshire, United States, in July 1944. The 44 countries at that conference sought to build a framework for economic cooperation to avoid a repetition of the competitive devaluations that had contributed to the Great Depression of the 1930s.

The example above was produced by the following HTML code:

< blockquote cite="http://www.imf.org/en/About"> The IMF, also known as the Fund, was conceived at a UN conference in Bretton Woods, New Hampshire, United States, in July 1944. The 44 countries at that conference sought to build a framework for economic cooperation to avoid a repetition of the competitive devaluations that had contributed to the Great Depression of the 1930s. </blockquote >

CSS Settings:

Properties of < blockquote > can be altered with CSS

Default CSS Settings:

blockquote {
display: block;
margin-top: 1em;
margin-bottom: 1em;
margin-left: 40px;
margin-right: 40px; }

HTML < hr > Tag

Definition, Usage, and Semantic Value(s)

The < hr > tag is used to separate content or define a break/change in an HTML page.

The < hr > tag has no end tag.

Example:

BCIT: Burnaby Campus
Address: 3700 Willingdon Ave, Burnaby, BC V5G 3H2





BCIT: Downtown Vancouver Campus
Address: 555 Seymour St, Vancouver, BC V6B 3H6

The red line that extends horizontally across the webpage is the result of the < hr > tag.

CSS Settings:

Properties of < hr > can be altered with CSS

Default CSS Settings:

hr {
display: block;
margin-top: 0.5em;
margin-bottom: 0.5em;
margin-left: auto;
margin-right: auto;
border-style: inset;
border-width: 1px; }
