CSS Scrollbars

When an element's content is larger than the amount of space assigned to it. We can use a CSS property called overflow to instruct the browser on what to do if an box's contents is larger than the box itself.

The overflow property can take one of the following values:

ValueDescription
visibleAllows the content to overflow the borders of its containing element.
hiddenThe content of the nested element is simply cut off at the border of the
containing element and no scrollbars is visible.
scrollThe size of the containing element does not change, but the scrollbars are
added to allow the user to scroll to see the content.
autoThe purpose is the same as scroll, but the scrollbar will be shown only if
the content does overflow.
