Web Usability

A Useful Website

A website I visit frequently: ESPN

I find this website extremely useful for keeping up with sports. The objective of the website is to give sports fans the latest scores, game schedules, analyses, and updates on sports teams they follow. When I access the homepage, I can see all the scores, updates, and latest developments in the sports I follow. The website design is very organized, all the latest scores are displayed at the very top, the tabs on the top allow me to select the sport I want to see, and all the latest news is shown in the middle. The website uses repetition to establish layouts and navigation locations. Furthermore, all contents are aligned neatly to ensure easy browsing.

Effective Use of Color

A website that uses color effectively: Waves Coffee

I find this website very aesthetically pleasing. The colors employed on this web page are light, soothing, and easy on the eyes. The web page also uses contrasting colors to separate the content from the background. Furthermore, darker colors are chosen to highlight important content in order to attract the visitor's attention.

Appealing Layout

A website with appealing Layout: Mercedes-Benz

I like the general layout of this website because of the color scheme, graphics, and alignment of content. The background and color scheme of the website accentuates the vehicles being displayed. In a way, the colors of the website gives off a luxurious vibe which rightfully describes the brand. As I scroll down the web page, all the contents are neatly aligned and easy to navigate. I can easily find which car models I am interested in or if there are any special offers available.

Essential Tips for Web Developers

  1. Do not use complex fonts that are illegible
  2. Do not autoplay video or audio files
  3. Do not cluster the web page with paragraphs of text
  4. Use pseudoclasses to highlight links
  5. Proofread! No spelling or grammatical mistakes
  6. Avoid tiny font sizes
  7. Avoid making long pages
  8. Do not use bright or harsh colors
  9. Make sure users can navigate the whole website
  10. Make sure the content can be quickly scanned by users
